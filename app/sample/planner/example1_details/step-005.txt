### 단계 5: 발급 수수료를 결제하고 주민등록등본을 다운로드한다.

1. **결제 페이지로 이동하기**
   - 정부24 웹사이트에서 주민등록등본 발급 신청이 완료된 후, 결제 페이지로 자동으로 이동합니다. 만약 자동으로 이동하지 않는 경우, 화면 상단의 '결제하기' 버튼을 클릭합니다.

2. **결제 수단 선택하기**
   - 결제 수단 선택 화면에서 사용 가능한 결제 방법(신용카드, 체크카드, 계좌이체 등) 중 하나를 선택합니다.
   - 예를 들어, 신용카드를 선택한 경우 '신용카드 결제' 버튼을 클릭합니다.

3. **결제 정보 입력하기**
   - 선택한 결제 수단에 따라 필요한 정보를 입력합니다.
     - **신용카드 결제 시**:
       - 카드 번호: 16자리 카드 번호를 입력합니다.
       - 유효기간: 카드의 유효기간(월/년)을 입력합니다.
       - CVC: 카드 뒷면의 3자리 보안 코드를 입력합니다.
     - **계좌이체 시**:
       - 은행 선택: 사용하고자 하는 은행을 선택합니다.
       - 계좌번호: 본인의 계좌번호를 입력합니다.
       - 비밀번호: 계좌 비밀번호를 입력합니다.

4. **결제 확인하기**
   - 입력한 결제 정보가 정확한지 다시 한 번 확인합니다.
   - '결제하기' 또는 '확인' 버튼을 클릭하여 결제를 진행합니다.

5. **결제 완료 확인하기**
   - 결제가 성공적으로 완료되면, 결제 완료 메시지가 화면에 표시됩니다.
   - 메시지 예시: "결제가 완료되었습니다. 주민등록등본을 다운로드 하실 수 있습니다."
   - 결제 영수증을 이메일로 받을 수 있는 옵션이 있을 경우, 이메일 주소를 입력하고 '영수증 받기' 버튼을 클릭합니다.

6. **주민등록등본 다운로드하기**
   - 결제 완료 후, '주민등록등본 다운로드' 버튼을 클릭합니다.
   - 다운로드가 시작되며, 파일이 컴퓨터에 저장됩니다.
   - 다운로드가 완료되면, 파일을 열어 주민등록등본이 정상적으로 발급되었는지 확인합니다.

### 필요한 준비물·입력값
- 신분증 (본인 확인을 위한)
- 결제 수단 (신용카드, 체크카드, 계좌이체 등)
- 결제 정보 (카드 번호, 유효기간, CVC 또는 계좌번호, 비밀번호 등)
