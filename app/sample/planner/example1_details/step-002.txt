### 단계 2: 회원가입 또는 로그인 한다.

1. **웹사이트 접속**
   - 웹 브라우저를 열고 주소창에 `www.gov.kr` 또는 `www.gov24.go.kr`을 입력한 후 Enter 키를 누릅니다.

2. **로그인 또는 회원가입 선택**
   - 정부24 웹사이트의 메인 페이지에서 오른쪽 상단에 위치한 "로그인" 버튼을 클릭합니다.
   - 로그인 페이지가 열리면, 이미 계정이 있는 경우 사용자 ID와 비밀번호를 입력합니다.
   - 만약 계정이 없다면, "회원가입" 링크를 클릭하여 회원가입 절차로 이동합니다.

3. **회원가입 절차 (계정이 없는 경우)**
   - 회원가입 페이지에서 다음 정보를 입력합니다:
     - **이메일 주소**: 유효한 이메일 주소를 입력합니다.
     - **비밀번호**: 보안 규칙에 맞는 비밀번호를 설정합니다 (대문자, 소문자, 숫자, 특수문자 포함).
     - **비밀번호 확인**: 동일한 비밀번호를 다시 입력합니다.
     - **이름**: 본인의 실명을 입력합니다.
     - **주민등록번호**: 주민등록번호를 입력합니다.
     - **전화번호**: 본인 확인을 위한 전화번호를 입력합니다.
   - 모든 필드를 정확히 입력한 후, "회원가입" 버튼을 클릭합니다.

4. **이메일 인증 (회원가입 시)**
   - 입력한 이메일 주소로 인증 메일이 발송됩니다. 이메일을 확인하고, 메일 내의 인증 링크를 클릭하여 인증을 완료합니다.

5. **로그인 (회원가입 후)**
   - 인증이 완료되면 다시 정부24 웹사이트로 돌아와 로그인 정보를 입력합니다.
   - 사용자 ID와 비밀번호를 입력한 후 "로그인" 버튼을 클릭합니다.

6. **로그인 확인**
   - 로그인 후, 화면 상단에 사용자 이름이 표시되면 로그인 성공입니다.
   - 만약 "비밀번호가 틀렸습니다" 또는 "계정이 존재하지 않습니다"라는 메시지가 나타나면, 입력한 정보가 정확한지 다시 확인합니다.

### 필요한 준비물·입력값
- 유효한 이메일 주소
- 본인 확인을 위한 주민등록번호
- 본인 명의의 전화번호
- 설정할 비밀번호 (보안 규칙 준수)

### [CHECKS]
- 로그인 후 사용자 이름이 화면에 표시되는지 확인합니다.
- 회원가입 시 이메일 인증 링크를 클릭 후, "인증이 완료되었습니다"라는 메시지가 나타나는지 확인합니다.
